# Registration Form

## Description
This project is a registration form built as part of the FreeCodeCamp curriculum.  
It focuses on creating structured forms, handling user input, and applying basic styling.

## Features
- Multiple input types (text, email, password)
- Radio buttons and checkboxes
- Proper label–input associations
- Built-in HTML form validation
- Clean and simple layout

## Tech Stack
- HTML5
- CSS3

## What I Learned
- Structuring forms using semantic HTML
- Linking labels correctly to inputs
- Using different input types effectively
- Applying basic validation with HTML attributes
- Improving layout and spacing with CSS

## How to Run
1. Clone or download the repository  
2. Open `index.html` in your browser  

## Project Status
Completed as part of FreeCodeCamp Responsive Web Design certification
